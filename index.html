Projeto MentalAid

Transtorno do Espectro Autista

  • Restrição interna para a compreensão de abstrações.
  • Dificuldade em compreender emoções

    • Objetivos

  • Explorar emoções básicas, comuns ao nosso cotidiano, de forma lúdica e interativa
  • Estimular o reforço de novas conexões cerebrais que estabeleçam as relações entre as expressões faciais e as emoções
  • Oferecer métricas significativas para o acompanhamento do quadro clínico da criança com TEA.

    • Threats and actions

    • Authentication & Authorization
    • Json Web Token para autenticação
    • User roles bem definidas para autorização
    • Broken Authentication
    • Senhas criptografadas com SALT e Hash (bcrypt.js)
    • Processo de login mais lento com o uso de criptografia
    • Limite de 250 requisições a cada 30 minutos (express-rate-limit)
    • Cross-Site Scripting (XSS)
    • JWT armazenado em HTTP Only Cookies
    • Sanitização de dados vindos do Client (xss-clean)
    • Set especial de HTTP Headers (helmet package)
    • NoSQL Query Injection
    • Schemas com validação criteriosa (Mongoose)
    • Sanitização de dados vindos do Client (express-mongoose-sanitize)
